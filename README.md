# MavenMarket_Report
Interactive Power BI dashboard using simulated financial and operational data for a fictional company

Project Overview
This project is an interactive Power BI dashboard developed for Maven Market, a fictional multinational grocery chain operating in the United States, Canada, and Mexico.

Dashboard Pages

1. Main Dashboard
Overview of the company's overall performance.
Is included:
-Total Revenue
-Total Profit
-Total Transactions
-Return Rate
-Monthly Revenue Trend
-Comparison between current and last month in terms of: Revenue, Profit, Transactions, Returns
-Top 30 Brands table

2. Map
Geographical visualization of transactions across the three operating countries: US, Canada and Mexico.
The map allows users to quickly identify transaction distribution by country.

3. Customer Detail
Customer analytics
Is included:
-Unique Customers
-Average Transactions per Customer
-Revenue per Customer
-Returns per Customer
Additional charts provide insights into the main customer categories.

4. Product Detail
Detailed analysis of product performance.
Is included:
-Top revenue-generating product
-Product transactions and revenue
-Revenue trend visualization
-Interactive product selector: users can select a product from the full product list and charts dynamically update to display the         selected product's revenue trend
-Products with the highest return rate
